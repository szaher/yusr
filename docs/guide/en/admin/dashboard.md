# Admin Dashboard

System-wide overview and management hub for academy administrators.

## Dashboard Overview

The admin dashboard provides comprehensive system analytics and quick access to management functions.

### System Statistics

**Enrollment Overview:**
- **Total Students** — All enrolled students
- **Active Students** — Currently active
- **Inactive Students** — Deactivated or graduated
- **Pending Applications** — Awaiting approval
- **Total Moderators** — Teaching staff count
- **Total Support Staff** — Support team count

**Organizational Structure:**
- **Levels** — Number of program levels
- **Classes** — Total classes across all levels
- **Groups** — Total memorization groups
- **Average Group Size** — Students per group

**Activity Metrics:**
- **Total Sessions** — All sessions conducted
- **This Week's Sessions** — Current week activity
- **Total Assignments** — All assignments created
- **Total Exams** — All exams administered

### Feature Overview

**System Health:**
- **System Status** — Operational, degraded, or down
- **Active Users** — Currently logged in
- **Recent Activity** — Last 24 hours
- **Error Rate** — System errors/issues

**Recent Activity Feed:**
- New student enrollments
- Moderator actions
- Support tickets
- System events
- Important alerts

## Quick Actions

### Administrative Tasks

**User Management:**
- **Review Applications** — Approve/reject pending enrollments
- **Create Moderator** — Add new teaching staff
- **Manage Users** — View, edit, deactivate users

**Organization:**
- **Create Level** — Add new program level
- **Create Class** — Add class to level
- **Create Group** — Set up new memorization group
- **Assign Students** — Place students in groups

**System:**
- **System Settings** — Configure platform
- **Feature Flags** — Enable/disable features
- **View Audit Log** — System activity history
- **Generate Reports** — Analytics and insights

## Enrollment Management

### Pending Applications

**Application Queue:**
- Number pending review
- Application details preview
- Quick approve/reject actions
- Bulk actions

**Application Review:**
1. Click **Review Applications**
2. View applicant information
3. Assess qualifications
4. Approve or reject
5. Assign to group (if approved)

### Enrollment Status

**System-Wide Control:**
- **Enrollment Open** — Accepting new applications
- **Enrollment Closed** — No new applications
- **Enrollment Limited** — Restricted to certain groups

**Setting Enrollment Status:**
1. Navigate to **Settings** > **Enrollment**
2. Toggle **Enrollment Status**
3. Set capacity limits (if limited)
4. Save changes

## Alerts and Notifications

### System Alerts

**Critical Alerts:**
- System errors
- Security issues
- Data integrity problems
- Service outages

**Administrative Alerts:**

- Moderator issues
- Student complaints
- Policy violations

**Activity Alerts:**

- Suspicious login attempts
- Data export requests
- System changes

### Managing Alerts

**Alert Settings:**
1. Navigate to **Settings** > **Notifications**
2. Configure alert thresholds
3. Set notification methods (email, SMS, in-app)
4. Choose alert recipients
5. Save preferences

## Dashboard Customization

### Widget Configuration

Personalize your admin dashboard:

1. Click **Customize Dashboard**
2. Add/remove widgets:
   - System stats
   - User overview
   - Recent activity
   - Pending tasks
   - Analytics charts
   - Alert feed
3. Resize and reorder widgets
4. Save layout

### Report Shortcuts

Create quick links to frequently accessed reports:

- Bookmark critical reports
- Schedule automatic report generation

### Mobile Features

**Mobile Limitations:**

# Creating and Managing Assignments

Assign Quran memorization, revision, Tajweed practice, and homework to your students.

## Assignment Overview

Use assignments to structure student work between sessions and provide additional learning opportunities.

### Assignment Types

1. **Quran Memorization** — New memorization homework
2. **Quran Revision** — Review previous memorization
3. **Tajweed Practice** — Focused Tajweed exercises
4. **Homework** — Islamic studies, reading, research, exercises

## Creating Assignments

### Basic Assignment Creation

To create a new assignment:

1. Navigate to **Assignments**
2. Click **Create Assignment**
3. Fill in assignment details:
   - **Type** — Select assignment type
   - **Title** — Assignment name (e.g., "Memorize Surah Al-Kahf verses 1-10")
   - **Description** — Detailed instructions
   - **Target** — Who receives it:
     - Entire group(s)
     - Specific students
     - All your students
   - **Due Date** — Deadline for completion
4. **Type-Specific Fields** (see below)
5. **Materials** — Attach resources (optional)
6. Click **Create Assignment**

### Quran Memorization Assignments

**Additional Fields:**
- **Quran Range** — Surah, verses, or pages to memorize
  - Select Surah and verse range
  - Or specify page numbers
- **Expected Preparation** — How students should prepare
- **Recitation Required** — Will they recite in next session?

**Example:**
```
Title: Memorize Surah Al-Baqarah 1-10
Type: Quran Memorization
Range: Surah Al-Baqarah, verses 1-10
Due: Next session (May 30)
Description: Memorize these verses with proper Tajweed. 
Focus on Makharij and Mad rules. Practice until fluent.
```

### Quran Revision Assignments

**Additional Fields:**
- **Revision Range** — Previously memorized portion to review
- **Focus Areas** — What to emphasize (accuracy, Tajweed, fluency)
- **Self-Assessment** — Should students self-check?

**Example:**
```
Title: Review Surah Al-Fatiha
Type: Quran Revision
Range: Surah Al-Fatiha (all verses)
Due: May 28
Description: Review complete Surah Al-Fatiha. 
Ensure perfect Tajweed. Be ready to recite from memory.
```

### Tajweed Practice Assignments

**Additional Fields:**
- **Tajweed Topic** — Specific rule to practice (Mad, Ghunnah, Qalqalah, etc.)
- **Practice Verses** — Specific verses containing these rules
- **Audio Resources** — Links to recitation examples

**Example:**
```
Title: Practice Noon Sakinah Rules
Type: Tajweed Practice
Topic: Ahkam (Izhar, Idgham, Iqlab, Ikhfa)
Verses: Surah Al-Baqarah 1-5
Due: May 27
Description: Listen to attached audio. Identify each Noon Sakinah rule. 
Practice reciting with correct application.
Materials: [Audio file - Sheikh Mishary recitation]
```

### Homework Assignments

**Additional Fields:**
- **Activity Type** — Reading, Watching, Writing, Research, Exercise
- **Submission Required** — Yes/No
- **Submission Type** — File upload, text response, listening confirmation

**Examples:**

**Reading Assignment:**
```
Title: Read Tafsir of Surah Al-Kahf
Type: Homework
Activity: Reading
Due: May 29
Description: Read the Tafsir (exegesis) of Surah Al-Kahf verses 1-10.
Materials: [PDF - Tafsir Ibn Kathir excerpt]
Submission: Confirm reading completion
```

**Video Assignment:**
```
Title: Watch Tajweed Lecture
Type: Homework
Activity: Watching
Due: May 28
Description: Watch lecture on Mad rules (40 minutes).
Materials: [YouTube link]
Submission: Answer 3 reflection questions (see attached)
```

**Writing Assignment:**
```
Title: Reflection on Surah Al-Fatiha
Type: Homework
Activity: Writing
Due: June 1
Description: Write a 250-word reflection on the themes of Surah Al-Fatiha.
Submission: Upload Word/PDF document
```

## Assignment Materials

### Attaching Resources

Add helpful materials to assignments:

**Supported Materials:**
- PDF documents
- Audio files (MP3, M4A)
- Video files (MP4, MOV) or links
- Images (JPG, PNG)
- External links (YouTube, websites)
- Worksheets and templates

**Adding Materials:**
1. In assignment creation form
2. Scroll to **Materials** section
3. Click **Add Material**
4. Choose:
   - **Upload File** — From your computer
   - **Add Link** — External URL
5. Add description for each material
6. Save assignment

**Material Best Practices:**
- Name files clearly (e.g., "Mad_Rules_Worksheet.pdf")
- Ensure files are accessible
- Test links before assigning
- Provide context for each resource
- Consider file sizes for mobile users

## Target Selection

### Assigning to Groups or Students

**Entire Group:**
- Select one or more of your groups
- All students in selected groups receive the assignment

**Specific Students:**
- Choose individual students
- Useful for differentiated instruction
- Remedial or advanced work

**All Your Students:**
- Quick option to assign to everyone you teach
- Use for general announcements or universal work

### Differentiated Assignments

Create multiple versions for different levels:

**Example:**
- Assignment A: Basic (beginner students)
- Assignment B: Intermediate (average students)
- Assignment C: Advanced (top performers)

Target each to appropriate students.

## Due Dates and Deadlines

### Setting Due Dates

**Options:**
- **Specific Date & Time** — Exact deadline
- **Next Session** — Due at next group meeting
- **Relative** — X days from creation

**Recommendations:**
- Give at least 3-5 days for memorization
- 2-3 days for revision
- 1-2 days for homework/reading
- Consider student schedules and commitments

### Late Submissions

**Policy Options:**
- Accept late work with note
- Accept with grade deduction
- Reject late submissions
- Extension requests allowed

Configure your preference in assignment settings or handle case-by-case.

## Grading Assignments

### Assignment Grading

For assignments requiring grading:

1. Navigate to assignment
2. Click **Grade Submissions**
3. For each student:
   - View their submission (file, text, confirmation)
   - Assign grade (0-100 or custom scale)
   - Provide written feedback
   - Mark as complete
4. Save grades

**Grading Criteria:**
- Completion (did they do it?)
- Quality (how well?)
- Timeliness (on time?)
- Effort demonstrated

### Feedback

**Effective Feedback:**
- Specific and actionable
- Balance positive and constructive
- Reference assignment criteria
- Encourage improvement

**Examples:**
- "Excellent understanding of Mad rules demonstrated. Practice Ghunnah duration."
- "Good effort. Review the Idgham rule (full merging without Ghunnah)."
- "Complete submission but rushed. Take time to reflect more deeply."

## Assignment Templates

### Saving Templates

Create reusable assignment templates:

1. Create an assignment with standard settings
2. Click **Save as Template**
3. Name the template
4. Template saved for future use

**Use Cases:**
- Weekly memorization homework
- Standard revision assignments
- Recurring Tajweed practice
- Common reading assignments

### Using Templates

To create assignment from template:

1. Click **Create Assignment**
2. Select **From Template**
3. Choose template
4. All fields pre-filled
5. Adjust as needed (dates, target, etc.)
6. Create assignment

**Saves time and ensures consistency.**

## Assignment Reminders

### Automatic Reminders

Students receive automatic reminders:
- 3 days before due date
- 1 day before due date
- On due date
- Daily for overdue (optional)

### Manual Reminders

Send manual reminder to students:

1. Open assignment
2. Click **Send Reminder**
3. Select students (all, incomplete only, specific students)
4. Optional: Add custom message
5. Send

## Editing and Deleting Assignments

### Editing Assignments

To modify an assignment:

1. Open assignment details
2. Click **Edit**
3. Make changes
4. Save
5. Students notified of updates (if significant changes)

**What Can Be Edited:**
- Title and description
- Due date (can extend)
- Materials (add/remove)
- Instructions

**What Cannot Be Edited:**
- Assignment type
- Target students (but can add more)

### Deleting Assignments

To delete an assignment:

1. Open assignment
2. Click **Delete Assignment**
3. Confirm deletion
4. Assignment removed
5. Students notified (assignment removed from their list)

> **Caution:** Deletion is permanent. Consider archiving instead if work has been submitted.

## Best Practices

### Effective Assignment Design

**Clear Instructions:**
- Specific, unambiguous requirements
- Step-by-step guidance
- Success criteria defined
- Examples provided (when helpful)

**Appropriate Difficulty:**
- Match student ability level
- Challenging but achievable
- Differentiate when needed
- Scaffold complex tasks

**Meaningful Work:**
- Aligned with learning goals
- Practical application
- Builds on previous learning
- Prepares for upcoming sessions

**Reasonable Workload:**
- Consider total workload
- Balance with other assignments
- Realistic time requirements
- Don't overwhelm students

### Assignment Frequency

**Recommended:**
- 1-2 assignments per week
- Always assign next memorization homework
- Periodic revision assignments
- Occasional enrichment homework

**Avoid:**
- Excessive daily assignments
- Busywork without purpose
- Overlapping deadlines
- Unrealistic completion timelines

---

**Assignments extend learning beyond your sessions. Design them thoughtfully, track completion diligently, and provide meaningful feedback.**

# Managing Sessions

Create, conduct, and manage live memorization sessions with your students.

## Creating Sessions

### Scheduling a New Session

To create a session:

1. Navigate to **Sessions**
2. Click **Create Session**
3. Fill in session details:
   - **Group** — Select which group
   - **Date** — Session date
   - **Time** — Start time (in your timezone)
   - **Duration** — Length in minutes (default: 60-90)
   - **Meeting Link** — Video conference URL (pre-filled from group settings)
   - **Type** — Regular, Review, or Exam
   - **Notes** — Optional session notes or agenda
4. Click **Create Session**

### Bulk Session Creation

Create multiple sessions at once:

1. Click **Create Sessions** > **Bulk Create**
2. Select group
3. Choose date range
4. Select recurrence:
   - Weekly (every week on same day)
   - Biweekly (every 2 weeks)
   - Custom pattern
5. Set time and duration
6. Meeting link (applies to all)
7. Preview sessions
8. Confirm creation

**Example:** Create all weekly sessions for a full term (12 weeks) in one action.

### Session Templates

Save session settings as templates:

1. Create a session with your preferred settings
2. Click **Save as Template**
3. Name the template
4. Use template for quick session creation

## Conducting Sessions

### Before the Session

**Preparation Checklist:**
1. ✓ Review student homework assignments
2. ✓ Check which students have prepared
3. ✓ Review previous session notes
4. ✓ Prepare any materials or resources
5. ✓ Test meeting link and audio/video
6. ✓ Join 5-10 minutes early

### Starting a Session

To begin a session:

1. Navigate to **Sessions**
2. Find today's session
3. Click **Start Session**
4. Session status changes to "In Progress"
5. Click **Join Meeting** to enter virtual classroom

**Session Start Actions:**
- Timer starts (optional)
- Students notified
- Attendance tracking enabled

### During the Session

**Session Activities:**
1. **Welcome and Opening** — Greet students, confirm audio/video
2. **Review Previous Material** — Quick recap
3. **Individual Recitations** — Call students one by one
4. **Take Notes** — Record mistakes, observations
5. **Provide Feedback** — Immediate correction and encouragement
6. **Group Activities** — Discussion, Q&A, group recitation
7. **Assign Homework** — Next memorization range
8. **Closing** — Dua, reminders, schedule confirmation

**Best Practices:**
- Keep students engaged
- Provide equal recitation time
- Positive reinforcement
- Clear, constructive feedback
- Note-taking during recitation
- Manage time effectively

### Ending a Session

To complete a session:

1. Conclude session activities
2. Click **End Session** or **Complete Session**
3. Session status changes to "Completed"
4. Proceed to attendance and grading

> **Note:** Completing a session doesn't automatically save attendance or grades. These must be entered separately.

## Taking Attendance

### Recording Attendance

After each session:

1. Open the completed session
2. Click **Take Attendance** or navigate to **Attendance** tab
3. Mark each student's attendance status

### Attendance Statuses

**Present** ✓
- Student attended on time
- Fully participated
- Mark as present

**Late** ⏰
- Student joined after session started
- Still participated
- Counts as attended but noted as late

**Absent** ✗
- Student did not attend
- No participation
- Unexcused absence

**Excused** 📋
- Pre-approved absence (leave request)
- Automatically marked if leave request approved
- Doesn't break attendance streak

### Bulk Attendance

**Quick Actions:**
- **Mark All Present** — Select all, then change exceptions
- **Mark from Previous Session** — Copy attendance pattern
- **Auto-Mark from Join Data** — If virtual classroom integration available

### Saving Attendance

1. Review all attendance marks
2. Ensure all students marked
3. Click **Save Attendance**
4. Attendance is recorded and locked
5. Students receive notification

> **Note:** Attendance can be edited later if mistakes were made. Click **Edit Attendance** from session details.

## Grading Students

### Session Grading

To assign grades for a session:

1. Open completed session
2. Navigate to **Grades** tab
3. For each student, enter:
   - **Grade** — Numeric score (0-100)
   - **Recitation Result** — Excellent, Very Good, Good, Needs Improvement
   - **Tajweed Score** — Separate Tajweed evaluation (0-100)
   - **Notes** — Specific feedback or observations
4. Click **Save Grades**

### Grading Criteria

**Recitation Performance (60% of grade):**
- Accuracy of memorization
- Fluency and confidence
- Error frequency and severity
- Correction of previous mistakes
- Retention since last review

**Tajweed Application (30% of grade):**
- Makharij (pronunciation points)
- Tajweed rules application
- Consistency
- Improvement

**Participation (10% of grade):**
- Preparation and readiness
- Engagement in session
- Respect and conduct
- Homework completion

### Grading Scale

Typical scale:

| Score | Grade | Description |
|-------|-------|-------------|
| 90-100 | A+ | Excellent, exceptional recitation |
| 85-89 | A | Very good, minor errors only |
| 80-84 | A- | Good, some errors corrected |
| 75-79 | B+ | Good, acceptable with corrections |
| 70-74 | B | Satisfactory, several corrections needed |
| 65-69 | C+ | Below expectations, significant errors |
| 60-64 | C | Needs improvement, many mistakes |
| <60 | F | Unsatisfactory, requires repetition |

**Recitation Results:**
- **Excellent** — 90-100: Ready to advance
- **Very Good** — 80-89: Can proceed with review
- **Good** — 70-79: Additional practice recommended
- **Needs Improvement** — <70: Must review and repeat

### Grading Tips

**Consistency:**
- Use same standards across all students
- Grade based on objective criteria
- Document reasoning for low/high grades

**Fairness:**
- Consider student's starting level
- Recognize improvement
- Don't compare students to each other
- Grade the performance, not the student

**Feedback:**
- Always provide written notes
- Specific, actionable feedback
- Balance correction with encouragement
- Highlight strengths and areas to improve

### Bulk Grading

For quick grading:

1. Use **Quick Grade** mode
2. Enter grades for all students on one screen
3. Use keyboard navigation (Tab, Enter)
4. Save all at once

## Rescheduling Sessions

### Changing Session Date/Time

To reschedule a session:

1. Open session details
2. Click **Reschedule**
3. Select new date and time
4. Update meeting link if changed
5. Add reason for reschedule
6. Click **Save Changes**
7. Confirm sending notifications to students

**Student Notification:**
- Email sent to all students
- In-app notification
- Updated session time in their dashboard
- Calendar invites updated

### Cancelling Sessions

To cancel a session:

1. Open session details
2. Click **Cancel Session**
3. Provide cancellation reason:
   - Moderator unavailable
   - Holiday
   - Technical issues
   - Emergency
   - Other (specify)
4. Optional: Create makeup session
5. Confirm cancellation
6. Students notified immediately

**Cancelled Session Effects:**
- Doesn't count toward attendance records
- No grades assigned
- Removed from active sessions
- Moved to cancelled sessions archive

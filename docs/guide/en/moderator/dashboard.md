# Moderator Dashboard

Your command center for managing memorization groups, tracking student progress, and conducting effective halaqat.

## Dashboard Overview

The moderator dashboard provides a comprehensive view of your groups and students:

### Key Information Widgets

**Your Groups Summary**
- **Total Groups** — Number of groups assigned to you
- **Total Students** — All students across your groups
- **Active Students** — Students currently enrolled
- **Groups List** — Quick access to each group

**Today's Overview**
- **Sessions Today** — Scheduled sessions for today
- **Upcoming Session** — Next session with countdown
- **Students to Review** — Pending memorization reviews
- **Assignments Due** — Assignments due today/this week

**Quick Statistics**
- **Overall Attendance Rate** — Average across all groups
- **Average Group Performance** — Mean session grades
- **Pending Reviews** — Student work awaiting your grading
- **Active Assignments** — Assignments currently in progress

### Group Cards

Each group you manage displays:

| Information | Description |
|-------------|-------------|
| **Group Name** | Level, class, and group number |
| **Student Count** | Active students in group |
| **Next Session** | Date and time of next meeting |
| **Attendance Rate** | Group attendance percentage |
| **Average Grade** | Mean performance score |
| **Quick Actions** | Start session, take attendance, view students |

## Student Alerts

### Attention Needed

Dashboard highlights students requiring attention:

**Academic Alerts:**
- **Low Grades** — Consistently scoring below 70%
- **Declining Performance** — Grades trending downward
- **No Progress** — No memorization logged in 2+ weeks
- **Failed Exams** — Below passing threshold

**Attendance Alerts:**
- **Chronic Absence** — Attendance below 70%
- **Consecutive Absences** — 3+ sessions in a row
- **Late Pattern** — Frequently arriving late
- **Broken Streak** — Lost long attendance streak

**Engagement Alerts:**
- **No Submissions** — Not completing assignments
- **Overdue Work** — Multiple overdue assignments
- **Inactive** — No platform activity in 7+ days

**Intervention Actions:**
- Contact student directly
- Involve parent/guardian
- Adjust memorization plan
- Provide extra support
- Report to administration

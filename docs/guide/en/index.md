# Welcome to Yusr Academy

Yusr Academy is a comprehensive web platform designed for managing Quran memorization circles (حلقات). The platform connects students, moderators, and administrators in a collaborative learning environment focused on Quran memorization, recitation excellence, and spiritual growth.

## What is Yusr Academy?

Yusr Academy provides a complete digital solution for traditional Quran memorization programs. Whether you're a student beginning your memorization journey, a moderator managing learning circles, or an administrator overseeing the entire program, Yusr Academy offers the tools you need to succeed.

## Key Features

- **Structured Memorization Plans** — Track progress through customizable memorization schedules with automatic pace calculation
- **Live Sessions** — Virtual classroom support with attendance tracking and session grading
- **Comprehensive Assessment** — Built-in exam system with multiple question types and automated grading
- **Progress Tracking** — Detailed analytics, milestones, badges, and achievement systems
- **Assignment Management** — Assign and track Quran memorization, revision, Tajweed practice, and homework
- **Bilingual Support** — Full Arabic and English interface with seamless language switching
- **Mobile-First Design** — Responsive layout with Progressive Web App (PWA) support for offline access
- **Communication Tools** — Support tickets, announcements, leave requests, and notifications

## User Roles

Yusr Academy supports four distinct roles, each with tailored features and permissions:

### Student
Students are the learners enrolled in memorization circles. They can:
- Track their memorization progress and review history
- Attend virtual sessions and view grades
- Complete assignments and take exams
- View achievements and compete on leaderboards
- Submit support tickets and leave requests

### Moderator
Moderators (معلم/معلمة) lead and manage one or more memorization groups. They can:
- Create and manage group sessions
- Track student attendance and assign grades
- Create memorization plans and log student reviews
- Assign coursework and exams
- Monitor student progress and award achievements

### Admin
Administrators oversee the entire academy system. They can:
- Manage enrollment applications and user accounts
- Create and organize levels, classes, and groups
- Configure system settings and feature flags
- View system-wide reports and analytics
- Manage memorization plan templates

### Support
Support staff handle student inquiries and administrative assistance. They can:
- View and respond to support tickets
- Manage leave requests
- Post system announcements
- Access user information for assistance

## Language Support

Yusr Academy is available in both Arabic and English. You can switch between languages at any time from the settings menu. All features, documentation, and user interfaces are fully localized.

## System Requirements

- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Stable internet connection for live sessions
- Optional: Install as PWA for offline access and mobile notifications

# إدارة المستخدمين

دليل لإدارة حسابات المستخدمين ومراجعة طلبات التسجيل وإنشاء حسابات المشرفين.

## مراجعة طلبات التسجيل

### صفحة الطلبات المعلقة

تعرض جميع طلبات التسجيل التي تنتظر مراجعتك:

| الطالب | البريد الإلكتروني | العمر | مستوى الحفظ | تاريخ التقديم | الإجراءات |
|--------|------------------|-------|-------------|---------------|-----------|
| أحمد محمد | ahmad@email.com | 16 | مبتدئ | 25 مايو | عرض / قبول / رفض |
| فاطمة علي | fatima@email.com | 14 | متوسط | 26 مايو | عرض / قبول / رفض |

### عرض تفاصيل الطلب

انقر على **عرض** لرؤية الطلب الكامل:

**المعلومات الشخصية:**
- الاسم الكامل
- تاريخ الميلاد والعمر
- الجنس
- البريد الإلكتروني
- رقم الهاتف
- البلد والمنطقة الزمنية

**معلومات الحفظ:**
- مستوى الحفظ الحالي
- مستوى التجويد
- الخبرة السابقة (وصف نصي)

**معلومات ولي الأمر:**
- اسم ولي الأمر
- بريد ولي الأمر الإلكتروني
- هاتف ولي الأمر

**التفضيلات:**
- اليوم المفضل للحلقات
- أي ملاحظات إضافية

### قبول طلب

عندما تقرر قبول طالب:

1. راجع جميع المعلومات
2. انقر على **قبول الطلب**
3. حدد:
   - **المستوى** — مبتدئ، متوسط، متقدم
   - **الفصل** — أي فصل دراسي
   - **المجموعة** — (اختياري) عيّن مباشرة أو لاحقاً
4. (اختياري) أضف **ملاحظات ترحيبية** — ستُرسل في البريد الإلكتروني
5. انقر على **تأكيد القبول**

**ما يحدث بعد القبول:**
- يتلقى الطالب بريداً إلكترونياً بالقبول
- يُنشأ حسابه رسمياً
- يمكنه تسجيل الدخول فوراً
- يُضاف للمستوى/الفصل المحدد

### رفض طلب

إذا لم تستطع قبول طالب:

1. انقر على **رفض الطلب**
2. اختر **سبب الرفض:**
   - المجموعات ممتلئة
   - العمر غير مناسب
   - معلومات ناقصة
   - متطلبات غير مستوفاة
   - أخرى (حدد)
3. أضف **رسالة للمتقدم** — اشرح السبب بلطف
4. (اختياري) اقترح **إعادة التقديم** — متى يمكنه التقديم مرة أخرى
5. انقر على **تأكيد الرفض**

**ما يحدث بعد الرفض:**
- يتلقى الطالب بريداً إلكترونياً بالرفض
- يُحذف الطلب من القائمة المعلقة
- يمكنه التقديم مرة أخرى لاحقاً

> **نصيحة:** كن مهنياً ولطيفاً في رسالة الرفض. اقترح بدائل إن أمكن.

## إدارة حسابات المستخدمين

### عرض جميع المستخدمين

صفحة **المستخدمون** تعرض جميع الحسابات:

**الفلاتر:**
- حسب الدور (طالب، مشرف، مدير، دعم)
- حسب الحالة (نشط، معلق، محظور)
- حسب المستوى/الفصل/المجموعة
- بحث بالاسم أو البريد الإلكتروني

### تفاصيل حساب المستخدم

انقر على أي مستخدم لعرض:

**معلومات الحساب:**
- الاسم، البريد، الهاتف
- الدور والصلاحيات
- تاريخ الإنشاء
- آخر تسجيل دخول

**حالة الحساب:**
- نشط — يمكنه استخدام النظام
- معلق — متوقف مؤقتاً
- محظور — ممنوع من الدخول

**الإحصائيات** (للطلاب):
- نسبة الحفظ
- معدل الحضور
- متوسط الدرجات

**المجموعات** (للطلاب):
- المستوى، الفصل، المجموعة
- تاريخ الانضمام

## إنشاء حسابات المشرفين

### خطوات الإنشاء

1. انتقل إلى **المستخدمون** > **إنشاء مشرف جديد**
2. املأ النموذج:

**المعلومات الشخصية:**
- الاسم الكامل
- البريد الإلكتروني (سيُستخدم لتسجيل الدخول)
- رقم الهاتف
- الجنس
- تاريخ الميلاد

**معلومات الحساب:**
- **كلمة المرور الأولية** — سيُطلب تغييرها عند أول تسجيل دخول
- **الصلاحيات:**
  - ✓ إدارة المجموعات
  - ✓ إنشاء حلقات
  - ✓ تقييم طلاب
  - ✓ إنشاء واجبات
  - ✓ إنشاء اختبارات
  - ✓ منح شارات يدوية
  - ✗ إدارة مستخدمين (للمديرين فقط)

**المجموعات المسندة:**
- (اختياري) عيّن مجموعات فوراً
- أو عيّن لاحقاً

3. انقر على **إنشاء الحساب**

**بعد الإنشاء:**
- يتلقى المشرف بريداً إلكترونياً ببيانات الدخول
- يُطلب منه تغيير كلمة المرور عند أول دخول
- يمكنه الوصول لمجموعاته فوراً

### إنشاء حسابات المديرين

مشابه لإنشاء المشرفين، لكن:

**الصلاحيات الإضافية:**
- ✓ إدارة المستخدمين
- ✓ إنشاء/حذف المستويات والفصول والمجموعات
- ✓ تكوين إعدادات النظام
- ✓ عرض جميع التقارير
- ✓ الوصول لسجل المراجعة

> **تحذير:** امنح صلاحيات المدير بحذر. الوصول الكامل للنظام!

## إجراءات الحساب

### تعليق حساب

لتعليق مؤقت:

1. افتح صفحة المستخدم
2. انقر على **تعليق الحساب**
3. اختر **السبب:**
   - انتهاك القوانين
   - طلب المستخدم
   - عدم النشاط
   - قيد التحقيق
4. حدد **المدة** — أيام، أسابيع، أو غير محدد
5. أضف **ملاحظة** (داخلية، لن يراها المستخدم)
6. أكّد التعليق

**التأثير:**
- لا يمكن تسجيل الدخول
- الحلقات والواجبات متوقفة
- البيانات محفوظة

### حظر حساب

للحظر الدائم:

1. افتح صفحة المستخدم
2. انقر على **حظر الحساب**
3. أدخل **السبب** (مطلوب)
4. أكّد الحظر

**التأثير:**
- ممنوع من تسجيل الدخول نهائياً
- لا يمكن إعادة التفعيل إلا من مدير
- البيانات محفوظة للسجلات

### إعادة تفعيل حساب

لحساب معلق أو محظور:

1. افتح صفحة المستخدم
2. انقر على **إعادة التفعيل**
3. أضف **ملاحظة** — سبب إعادة التفعيل
4. أكّد

الحساب يعود للحالة النشطة فوراً.

### إعادة تعيين كلمة المرور

إذا نسي مستخدم كلمة مروره:

1. افتح صفحة المستخدم
2. انقر على **إعادة تعيين كلمة المرور**
3. اختر:
   - **إرسال رابط إعادة التعيين** — للبريد الإلكتروني
   - **تعيين كلمة مرور مؤقتة** — تحددها أنت
4. إذا اخترت المؤقتة:
   - أدخل كلمة المرور
   - ✓ **إجبار التغيير عند الدخول التالي**
5. أكّد

### حذف حساب

> **تحذير شديد:** الحذف نهائي ولا يمكن التراجع عنه!

لحذف حساب (نادر جداً):

1. افتح صفحة المستخدم
2. انقر على **حذف الحساب**
3. ستظهر رسالة تحذير
4. أدخل **سبب الحذف** (مطلوب للسجلات)
5. اكتب **DELETE** بالأحرف الكبيرة لتأكيد
6. انقر على **حذف نهائياً**

**ما يُحذف:**
- الحساب
- المعلومات الشخصية
- (اختياري) السجلات الأكاديمية

**ما يُحفظ:**
- إحصائيات مجهولة المصدر
- سجلات النظام

## إدارة حالة التسجيل

### تعطيل التسجيل

لإغلاق التسجيل مؤقتاً:

1. انتقل إلى **الإعدادات** > **التسجيل**
2. عطّل **السماح بالتسجيل الجديد**
3. أضف **رسالة** — تُعرض للزوار
4. احفظ

**مثال رسالة:**
"التسجيل مغلق مؤقتاً. سيُفتح مجدداً في 1 سبتمبر 2026."

### إعدادات التسجيل

**الموافقة:**
- ✓ **موافقة يدوية** — المديرون يراجعون كل طلب
- ✗ **موافقة تلقائية** — قبول فوري (غير منصوح به)

**متطلبات:**
- الحد الأدنى للعمر: 10 سنوات
- الحد الأقصى للعمر: لا يوجد
- ✓ معلومات ولي الأمر مطلوبة (لمن هم دون 18)
- ✓ تأكيد البريد الإلكتروني مطلوب

**الحدود:**
- الحد الأقصى للطلاب: 500
- تحذير عند الوصول: 80% (400 طالب)

## البحث المتقدم

أدوات بحث قوية للعثور على مستخدمين:

**البحث الأساسي:**
- بالاسم
- بالبريد الإلكتروني
- برقم الهاتف

**البحث المتقدم:**
- حسب نطاق الأعمار
- حسب تاريخ التسجيل
- حسب آخر تسجيل دخول
- حسب معدل الحضور
- حسب نسبة الحفظ
- حسب عدد الشارات

**أمثلة:**
- "طلاب لم يسجلوا دخول منذ 30 يوماً"
- "طلاب حفظوا أكثر من 50%"
- "طلاب حضور أقل من 60%"
